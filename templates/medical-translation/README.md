# 医患语言无障碍翻译助手

> **Blueprint** — 已在长庚医院私有化部署

## 快速开始

```bash
npx mland-cli add medical-translation --hospital=长庚医院
cd medical-translation
docker compose up -d
```

## 目录结构

```
medical-translation/
├── miniprogram/     # 微信小程序前端
├── server/          # API Gateway
├── .env.example
└── docker-compose.yml
```
